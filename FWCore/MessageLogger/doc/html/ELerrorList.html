Examining Error Message History

The user attaches an ELerrorList destination. This will apply all the ordinary filtering to each error messages, and insert any message that passes the filtering onto a std:list of ErrorObjs.

The user code is free to examine and modify (including clearing) this list.

Example program of the use of ELerrorList: testELerrorList.cc
